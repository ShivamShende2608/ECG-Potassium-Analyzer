# ECG Potassium Analyzer 🫀⚡

AI-powered mobile application for non-invasive potassium disorder 
detection from ECG images in CKD patients.

## 🔬 Features
- 3-Lead ECG capture (Clean ECG + 12-Lead modes)
- EfficientNetB4 deep learning model (91%+ accuracy, AUC 0.97+)
- GradCAM visual explainability
- 7-parameter XAI engine (T, P, QRS, ST, U, PR, QT)
- Real-time grid removal preprocessing

## 🛠️ Tech Stack
- **Mobile**: React Native (Expo)
- **Backend**: Python FastAPI
- **Model**: PyTorch EfficientNetB4 (timm)
- **Dataset**: PhysioNet (25,000+ ECG images)
